# タスク完了時のチェックリスト

コードの変更やタスクが完了したら、以下のステップを実行してください：

## 1. コードフォーマット
```bash
cargo fmt
```

## 2. リンターチェック
```bash
cargo clippy
```
警告が出た場合は修正を検討してください。

## 3. テストの実行
```bash
cargo test
```
すべてのテストがパスすることを確認してください。

## 4. ビルドの確認
```bash
cargo build
```
プロジェクトがエラーなくビルドできることを確認してください。

## 5. 実行確認（必要に応じて）
```bash
cargo run -- <test_value>
```
実際の動作を確認してください。

## 推奨の実行順序
```bash
cargo fmt && cargo clippy && cargo test && cargo build
```

このワンライナーで一連のチェックを実行できます。
