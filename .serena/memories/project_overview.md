# dechex - プロジェクト概要

## プロジェクトの目的
このプロジェクトは、10進数と16進数の相互変換を行うコマンドラインツールです。
- 16進数（0xプレフィックス付き）を10進数に変換
- 10進数を16進数（0xプレフィックス付き）に変換

## テクノロジースタック
- **言語**: Rust (Edition 2024)
- **主な依存関係**:
  - `clap` (v4) - コマンドラインパーサー（derive機能を使用）
  - `regex` (v1) - 正規表現処理

## プロジェクト構造
```
dechex/
├── Cargo.toml       # プロジェクト設定と依存関係
└── src/
    └── main.rs      # メインアプリケーションコード（Converter実装とテストを含む）
```

## エントリーポイント
- メインアプリケーション: `src/main.rs`
  - `main()` 関数がCLI引数を解析し、`Converter`を使用して変換を実行
